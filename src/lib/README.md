# Library (lib)

This directory contains utility functions, configurations, and helper modules.

## Purpose
- Store shared utilities and helper functions
- Configuration files for services (database, APIs, etc.)
- Constants and enums
- Custom hooks

## Structure

- Use TypeScript for type-safe utilities
- Document complex functions
